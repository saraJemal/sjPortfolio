Online Exam Web Service

The main goal of Online Exam Web Service (OEWS) is to effectively evaluate the shortlisted applicants in staff recruitment thoroughly through a totally automated system that not only reduce the required time but also obtain fast and accurate results....

Online Exam Web Service’s some features are stated as follows:


  • Saves the exams and users information in a database.
  • Enables exams only for scheduled period of time.
  • Displays random questions for students or applicants.
  • Auto submits the exam upon expiration of allowed time.
  • Auto marks the attempted exam and produce the reports of the results.
  • Prevent exam refreshing and resubmission using session expiration and database.

Employee Work Hour Tracker

The main aim of this application is to keep track of attendance and employees’ wages. Attendance tracking based on GPS location is enabled to avoid any attempt to add proxy attendance. The administration's role is to add new employee details and provide the employee with the respective user name and Password to the Employee so that access to the app is enabled...

Administrative staff will be able to view the GPS location of each employee by entering the employee’s user name and date. Administrative staff will also be able to view the latitude and longitude of the employee’s GPS location as well as the number of logins and logouts on a particular day. The hours will contribute to the calculation of the hourly wage that will be displayed on the mobile apps. The employee will log in with his or her image captured with GPS location. This information will be sent to the Administration. Upon login, the GPS location will be automatically tracked by the system after every 10 minutes. Hourly wages will be based on the number of hours worked on a particular day.

Login Page

This page is a general login page. All users namely: Admin, HR, and Applicant can login to their respective page using their own email and the password that stored in a database when they are registered. When a user clicks a login button shown on the right-top corner of the main page of the system, it leads the user to this page which asks for email and password of the user. If a user enters his/her email and password, the system checks for their match in the data base and under which role they are, then it leads the user to his/her respective role page namely: Admin Page, HR Page, and Applicant Page.

Admin Page

Admin registers the detail of users and generates their passwords. And he/she can add/delete/update users and view their activities. Overall management of users are carried out by admin.

Hr Page

HR can add/delete/update schedules, time limited questions to the database. In this page, first an HR adds schedules which contains Subjects, total number of questions, total marks, given duration, and the range of date in which the exam is available online for an applicant to attempt it. There are also links that help to add/delete/edit multiple choices questions, to upload multiple choices questions from excel file, and to review details of results of applicants that have attempted the exams.

Aplicant Page

Upon successful login an applicant views exam instruction and schedules of all subjects. Then he/she checks schedule of the subject they intend to attempt by selecting from given select box and pressing check Availability button. If the exam is available, the system enable start exam button. By clicking the start exam button, he/she can attempt the question of that particular subject. After answering questions and clicking submit button, the result and other information of it will be displayed on the screen instantly. Then he/she can logout from the system by pressing logout button.
